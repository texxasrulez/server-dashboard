# i18n

## Files
- `includes/i18n.php` — PHP helpers: `__('key')`, `_e('key')`
- `assets/js/i18n.js` — JS loader: `I18N.t(key)`, `data-i18n="key"`
- `assets/i18n/en.json` — Base English strings
- `scripts/i18n_scan.php` — Scans code for keys and outputs a JSON skeleton

## Wiring
- In your global HTML template, set `<html lang="<?= h(DASH_LOCALE) ?>">`.
- Include `<?php require_once __DIR__.'/i18n.php'; ?>` early (e.g., in your page header include).
- Add `<script src="assets/js/i18n.js"></script>` before other page scripts.
